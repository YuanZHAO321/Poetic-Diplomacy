# 赋诗之礼：规则与原文依据

本文档是赋诗言志的"游戏规则书"。每条规则都给出《左传》原文，写作时可直接引用。

## 一、场合与仪程

赋诗不是随时可发，它镶嵌在礼仪的固定时点上：

**聘问全程**（以昭公二年韩宣子聘鲁、聘卫为范本）：
郊劳（主国迎于郊，客可辞）→ 致馆（安排宾馆，客可辞谦）→ **享**（正式宴礼，庄重，主国之君设）→ **宴**（享礼毕后的私宴，"既享，宴于季氏"）→ 饯（临行郊外送别）。

- **享礼中赋诗**在"礼终乃宴"之后：昭公元年郑伯享赵孟，"赵孟为客，礼终乃宴。穆叔赋《鹊巢》……"
- **饯行赋诗**最自由，是观志名场面的高发地：郑六卿饯韩宣子于郊（昭十六）、垂陇之享（襄二十七）。
- **乞师庭中**亦可赋诗定夺：申包胥哭于秦庭七日，"秦哀公为之赋《无衣》，九顿首而坐，秦师乃出"（定四）。

**享燕的规格本身就是外交语言**：昭公元年子皮为赵孟备"五献之笾豆"，赵孟辞而"乃用一献"——先以《瓠叶》（"酌言尝之"，薄酒之诗）预告自己只要一献之礼。用诗预告礼数，是高段位操作。

**谁来"赋"**：卿大夫亲自"赋"（口诵其辞），乐工代唱称"歌"/"使工为之诵"。让乐工唱有居高临下之意（天子、国君对臣下），也可用来传递不便亲口说的讽刺（卫献公使大师歌《巧言》之卒章以辱孙文子，襄十四——结果引发叛乱，本技能列为反面教材）。

**拜的等级**（由轻到重）：拜 → 再拜 → 降拜（降阶而拜，示不敢当堂受礼）→ 再拜稽首（叩首至地，最重）。
- 文公三年：晋侯飨公赋《菁菁者莪》（"既见君子，乐且有仪"，喻鲁君为人材），庄叔立刻让鲁文公**降拜**："小国受命于大国，敢不慎仪！"晋侯降阶辞让，再登堂成拜。
- 僖公二十三年：秦穆公赋《六月》（尹吉甫佐天子——喻重耳将匡辅王国），赵衰喝令："重耳拜赐！"公子**降，拜，稽首**——因为这首诗等于承诺送他回国为君。
- 听出对方的诗抬举过高时，正确动作是离席避让："公赋《南山有台》，武子**去所**，曰：'臣不堪也。'"（襄二十）

## 二、断章取义（合法引用规则）

> 卢蒲癸曰："**赋诗断章，余取所求焉**，恶识宗！"（襄二十八）

取一章甚至一句，只用其字面所需之义，完全无视原诗本义与上下文——这是全体玩家公认的规则，不是曲解。典型：

| 原诗本义 | 断章后的外交义 |
|---|---|
| 《野有蔓草》男女野外相遇之情诗 | "邂逅相遇，适我愿兮"→ 与您相会真是我的心愿（襄二十七、昭十六两见，皆获好评） |
| 《摽有梅》女子催婚 | "迨其吉兮"→ 请趁时机赶快行动（范宣子借以催鲁出兵伐郑，襄八） |
| 《匏有苦叶》女子待嫁渡口 | "深则厉，浅则揭"→ 我决意渡河伐秦（穆叔赋之，叔向退而具舟，襄十四） |
| 《褰裳》情人间赌气 | "子不我思，岂无他人"→ 你若不管我，我就投靠别国（子大叔警告韩宣子，昭十六） |

注意断章以**章**为单位截取："赋《载驰》之四章"、"赋《角弓》"（全篇）、"赋《绿衣》之卒章"。指定章次可以精确规避原诗中不合用的句子。

## 三、歌诗必类（相称原则）

> 晋侯与诸侯宴于温，使诸大夫舞，曰："**歌诗必类**！"齐高厚之诗不类。荀偃怒，且曰："**诸侯有异志矣**！"使诸大夫盟高厚，高厚逃归。……（诸侯）同盟曰："同讨不庭。"（襄十六）

诗与场合、身份、当下议题不相称，立即被解读为政治信号——轻则失礼，重则如高厚：一支舞歌不类，齐国被判定怀有二心，成为诸侯联军讨伐的口实。评赋时"类不类"是第一判据。

失类的极端案例：垂陇之享，伯有赋《鹑之贲贲》（"人之无良，我以为君"——闺房怨刺之诗），赵孟当场冷评"**床笫之言不逾阈**，况在野乎？非使人之所得闻也"，事后断言"**伯有将为戮矣！诗以言志**，志诬其上而公怨之……幸而后亡"（襄二十七）。用诗骂自家国君给外人听 = 自曝将叛，等于宣判自己死刑。

## 四、等级限度（僭越红线）

乐与诗各有专属等级，僭用即大失礼。两个标杆案例：

**穆叔如晋**（襄四）——精确的"该拜什么不拜什么"：
> 金奏《肆夏》之三，不拜。工歌《文王》之三，又不拜。歌《鹿鸣》之三，三拜。……对曰："三《夏》，**天子所以享元侯也**，使臣弗敢与闻。《文王》，**两君相见之乐也**，使臣不敢及。《鹿鸣》，**君所以嘉寡君也**，敢不拜嘉？《四牡》，君所以劳使臣也，敢不重拜？《皇皇者华》，君教使臣曰'必咨于周'……臣获五善，敢不重拜？"

**甯武子不答《湛露》《彤弓》**（文四）——以佯装无知拒绝僭礼：
> 公与之宴，为赋《湛露》及《彤弓》。**不辞，又不答赋**。使行人私焉。对曰："臣以为**肄业及之**也（还以为是乐工练习呢）。昔诸侯朝正于王，王宴乐之，于是乎赋《湛露》，则天子当阳，诸侯用命也。诸侯敌王所忾而献其功，王于是乎赐之彤弓一……今陪臣来继旧好，君辱贶之，**其敢干大礼以自取戾**？"

速查表：
| 诗/乐 | 专属场合 | 卿大夫遇之 |
|---|---|---|
| 《肆夏》三章（金奏） | 天子享元侯 | 不拜，弗敢与闻 |
| 《文王》《大明》《绵》 | 两君相见 | 不拜（但答赋中借单章颂德可通融，如季武子赋《绵》之卒章颂韩宣子） |
| 《湛露》《彤弓》 | 天子宴诸侯、赐功臣 | 不辞不答，佯为肄业 |
| 《鹿鸣》《四牡》《皇皇者华》 | 君嘉宾、劳使臣 | 每章拜谢 |
| 国风情诗 | 断章言好，人人可用 | 正常答赋 |

例外通道：小国之君对大国执政赋高格之诗是"尊之"，对方以"不堪"谦辞即可收下好意（襄十九范宣子赋《黍苗》比晋侯为召伯，季武子兴，再拜稽首）。

## 五、答赋义务与沉默的语义学

被赋诗必须回应。回应方式构成一个信号谱：

| 回应 | 含义 | 例 |
|---|---|---|
| 答赋相称之诗 | 接受并推进对话 | 文十三棐之宴四诗成一场完整谈判 |
| 拜 / 降拜 / 稽首 | 接受且感戴，等于签字 | 重耳稽首受《六月》（僖二十三） |
| "（某）不堪也" + 去所/避让 | 谦不敢当（仍是善意） | 赵孟答《鹊巢》"武不堪也"（昭元）；季武子去所（襄二十） |
| 不辞又不答赋 + 事后私下解释 | 守礼拒收僭格之礼 | 甯武子（文四） |
| 弗知，不答 | 无学失礼，判为将亡 | 宋华定不识《蓼萧》："必亡。宴语之不怀，宠光之不宣，令德之不知，同福之不受，将何以在？"（昭十二）；庆封不知《相鼠》《茅鸱》（襄二十七经传、二十八） |

设计场景时务必让"沉默"有明确归属——是拒、是谦、还是不学。

## 六、赋诗即承诺（言语行为的效力）

诗一出口即生效力，等同口头条约，《左传》从无赖账记录：

- 秦哀公赋《无衣》（"王于兴师，修我戈矛，与子同仇"）——申包胥九顿首而坐，**秦师乃出**（定四）。赋诗就是出兵令。
- 晋侯本囚卫侯，国子赋《辔之柔矣》（逸诗，宽政之喻）、子展赋《将仲子兮》（"人之多言，亦可畏也"——诸侯舆论可畏），**晋侯乃许归卫侯**（襄二十六）。两首诗救回一位国君。
- 叔孙穆子赋《匏有苦叶》，叔向不待多言"**退而具舟**"（襄十四）——听懂即执行。
- 楚灵王享郑伯赋《吉日》（天子田猎之诗），子产会后即"**乃具田备**"（昭三）——诗就是行程通知。
- 荣成伯赋《式微》（"式微式微，胡不归"），鲁襄公**乃归**（襄二十九）。

因此拟赋时要提醒用户：赋出的诗对方一旦拜受，我方即受约束。

## 七、诗以观志（听诗识人术）

"《诗》以言志"——赋什么诗暴露什么心术，高手据此预言个人与国家命运：

- **垂陇观志**（襄二十七）：赵孟请郑七卿各赋，逐一点评——子展《草虫》（"未见君子，忧心忡忡"尊晋如君子）→"民之主也"；印段《蟋蟀》（"好乐无荒"）→"保家之主也"；公孙段《桑扈》（"匪交匪敖"）→"福将焉往？"；伯有《鹑之贲贲》→ 预言被杀（后果验）。
- **令尹之志**（昭元）：楚令尹围享赵孟赋《大明》之首章（"明明在下，赫赫在上"——文王受命！），赵孟答《小宛》之二章（"各敬尔仪，**天命不又**"——委婉警告天命不会再来）。事后赵孟问叔向"令尹自以为王矣"，叔向断："不义而强，**其毙必速**。"
- **季札观乐**（襄二十九）：闻歌各国之风而论其国运——"为之歌《郑》，曰：'美哉！其细已甚，民弗堪也，**是其先亡乎**！'"这是观志术的极致：从乐风听出政风。

评赋模式的输出就是复现这套推理：诗中何句 → 泄露何志 → 推出何种结局。

## 八、称谓与套语速查

| 语境 | 用语 |
|---|---|
| 自称己国/己君 | 敝邑、寡君（对外称自己国君）、下臣、陪臣（臣之臣对他国君）、不腆（不丰厚，谦） |
| 敬称对方 | 君、吾子、执事（不敢直指其人）、大国、二三君子 |
| 承接好意 | 敢不承命、敢拜（其）辱、辱贶之、君之惠也、敢不拜嘉 |
| 谦拒 | （名）不堪也、弗敢与闻、不敢及、其敢干大礼以自取戾 |
| 施压收尾 | 唯君图之、敢私布之、不敢不告 |
| 主持人开场 | 请皆赋以卒君贶，（某）亦以观（某国）之志（仿韩宣子、赵孟） |

对话中卿大夫自称用名（赵武自称"武"，韩起自称"起"，士匄自称"匄"，叔孙豹自称"豹"）——这是文言身份感的关键细节。
