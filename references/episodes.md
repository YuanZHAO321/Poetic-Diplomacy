# 《左传》赋诗事例全库

《左传》全书赋/歌/诵《诗》共 83 处，合并为 30 个独立事件，本库全收，按编年排列。**加粗**者为最具范式价值的十大名场面，演礼、拟赋优先援引。

体例：〖年份〗场合 → 赋诗链 → 结局 → 可援引之点。

## 目录

- 甲编：外交赋诗（核心 19 事）
- 乙编：源头与边缘（民歌时政、宫廷内政、军旅，11 事）

## 甲编：外交赋诗

### **1.〖僖公二十三年〗秦穆公享公子重耳——乞纳之赋**
流亡的重耳到秦国。子犯自认"吾不如衰之文也"，让赵衰陪席。**公子赋《河水》**（逸诗，取"朝宗于海"，喻晋归服于秦），**穆公赋《六月》**（尹吉甫佐天子征伐，喻重耳将匡辅王国——等于承诺送他回国即位）。赵衰喝令"重耳拜赐"，公子降拜稽首，公降一级而辞。
→ 可援引：求援—许诺的标准两诗结构；重礼级答谢（降拜稽首）；文臣陪席的价值。

### 2.〖文公三年〗晋侯飨鲁文公——大国抚小国
晋侯飨公，**赋《菁菁者莪》**（"既见君子，乐且有仪"，乐育人材，喻嘉鲁君）。庄叔以公降拜："小国受命于大国，敢不慎仪！君贶之以大礼，何乐如之。"晋侯降辞，登，成拜。**公赋《嘉乐》**（即今本《假乐》，"宜民宜人，受禄于天"颂晋侯）。
→ 可援引：小国受高格礼遇的全套身段（降拜+归美于大国）。

### **3.〖文公四年〗卫甯武子不答《湛露》《彤弓》——拒僭之默**
鲁文公宴甯武子，为赋《湛露》及《彤弓》（皆天子宴诸侯之乐）。甯武子**不辞又不答赋**，事后使行人私问，答："臣以为肄业及之也……今陪臣来继旧好，君辱贶之，其敢干大礼以自取戾？"
→ 可援引：以沉默守礼的最高范本；"佯装乐工练习"的外交台阶。

### 4.〖文公七年〗荀林父赋《板》之三章——同僚苦谏
先蔑将出使秦国迎立公子雍，荀林父劝阻不听，**为赋《板》之三章**（"我言维服，勿以为笑"——我的话是正经忠告）。又弗听。后先蔑果然被牵连奔秦。
→ 可援引：赋诗作最后一谏；不听诗谏的下场。

### **5.〖文公十三年〗棐之宴：郑请鲁为之请平于晋——四诗谈判**
郑穆公与鲁文公宴于棐。**子家赋《鸿雁》**（"爰及矜人，哀此鳏寡"——郑是可怜的孤弱者，求鲁怜恤奔走）。季文子答"寡君未免于此"（我们也苦），**赋《四月》**（"四月维夏……胡宁忍予"——行役逾时思归，婉拒跑腿）。**子家赋《载驰》之四章**（"控于大邦，谁因谁极"——小国有难必须求告大国，你不帮谁帮）。**文子赋《采薇》之四章**（"岂敢定居？一月三捷"——好吧，不敢安居，答应奔走）。郑伯拜，公答拜。
→ 可援引：完整的请求—婉拒—加码—允诺四步谈判；是"拟赋"多轮设计的第一范本。

### 6.〖成公九年〗季文子如宋致女——穆姜答《绿衣》
季文子送鲁女嫁宋归来复命，公享之。文子**赋《韩奕》之五章**（蹶父嫁女"孔乐韩土"——宋国是好归宿，姑娘安顿好了）。鲁宣公遗孀穆姜自房中出，再拜："大夫勤辱，不忘先君以及嗣君，施及未亡人。"**又赋《绿衣》之卒章**（"我思古人，实获我心"——你深得我心）。
→ 可援引：赋诗用于姻亲外交与"复命"汇报；女性赋诗的罕见实例。

### **7.〖襄公四年〗穆叔如晋：三拜《鹿鸣》——等级教科书**
晋侯享穆叔：金奏《肆夏》之三，不拜；工歌《文王》之三，又不拜；歌《鹿鸣》之三，三拜。韩献子使人问故，穆叔条对："三《夏》天子所以享元侯……《文王》两君相见之乐……《鹿鸣》君所以嘉寡君也，敢不拜嘉？"
→ 可援引：乐诗等级表的原始出处；"该拜什么"的操作规范。

### 8.〖襄公八年〗范宣子聘鲁催兵——《摽有梅》
晋范宣子来聘，告将用师于郑。公享之，**宣子赋《摽有梅》**（"求我庶士，迨其吉兮"——趁时机快来，催鲁及时出兵）。季武子答："谁敢哉！……欢以承命，何时之有？"**赋《角弓》**（"兄弟婚姻，无胥远矣"——晋鲁兄弟勿相远）。宾将出，**武子赋《彤弓》**（天子赐功臣之弓——追述晋文公受赐、恭维晋国世为盟主）。宣子答以城濮受弓故事："敢不承命？"君子以为知礼。
→ 可援引：催促盟军的委婉句式；《彤弓》由小国用来颂大国霸主身份的特殊合法用法。

### **9.〖襄公十四年〗戎子驹支自辩——《青蝇》退谗**
晋范宣子当众数落姜戎首领驹支泄密。驹支历陈戎人从晋百役之功后，"**赋《青蝇》而退**"（"营营青蝇……谗人罔极"——你听信谗言了）。宣子当场道歉（"辞焉"），请他与会成礼。
→ 可援引：弱者以诗自辩反将一军；赋诗跨族群通用（戎首亦谙《诗》）。

### 10.〖襄公十四年〗穆叔赋《匏有苦叶》——诗即军令
诸侯之师伐秦，至泾水不肯渡。叔向见叔孙穆子，**穆子赋《匏有苦叶》**（"深则厉，浅则揭"——无论深浅都要渡）。**叔向退而具舟**，鲁人、莒人先济。
→ 可援引：赋诗即承诺/命令的效力；听懂即执行的默契。

### 11.〖襄公十六年〗温之会：歌诗必类
晋侯与诸侯宴于温，使诸大夫舞，令"歌诗必类"。**齐高厚之诗不类**，荀偃怒："诸侯有异志矣！"使诸大夫盟高厚，高厚逃归。诸侯同盟"同讨不庭"，伐齐之局遂成。
→ 可援引："类"原则与失类的最重后果；评赋判据之首。

### 12.〖襄公十六年〗穆叔如晋乞师——《圻父》《鸿雁》
齐师压境，穆叔如晋告急。见中行献子，**赋《圻父》**（即《祈父》："祈父，予王之爪牙，胡转予于恤？"——你身为司马为何让我陷于忧患，责其失职）。献子谢罪："偃知罪矣！"见范宣子，**赋《鸿雁》之卒章**（"哀鸣嗷嗷……谓我劬劳"——鲁在哀鸣）。宣子答："匄在此，敢使鲁无鸠乎？"
→ 可援引：乞师双诗（一责一哀）；对不同执政选不同修辞。

### 13.〖襄公十九年〗季武子如晋拜师——《黍苗》《六月》
鲁借晋师胜齐，季武子赴晋拜谢。范宣子**赋《黍苗》**（"芃芃黍苗，阴雨膏之。悠悠南行，召伯劳之"——晋侯如召伯膏泽诸侯）。季武子兴，再拜稽首："小国之仰大国也，如百谷之仰膏雨焉！"**赋《六月》**（比晋侯于尹吉甫，匡王国者）。
→ 可援引：致谢外交的标准配诗；"稽首"用于谢师。

### 14.〖襄公十九年〗穆叔会范宣子——《载驰》之四章
齐晋既平，穆叔仍忧齐，见叔向**赋《载驰》之四章**（"控于大邦"——鲁仍需晋援）。叔向："肸敢不承命。"穆叔："齐犹未也，不可以不惧。"乃城武城。
→ 可援引：同一首诗（载驰四章）在不同年份复用同一义项——经典义项已固化可查。

### 15.〖襄公二十年〗季武子聘宋——《常棣》《鱼丽》《南山有台》
季武子如宋报聘，褚师段迎之受享，**赋《常棣》之七章以卒**（"妻子好合……兄弟既翕"——鲁宋如兄弟）。宋人重贿之。归而复命，公享之，**武子赋《鱼丽》之卒章**（"物其有矣，维其时矣"——聘问成果丰而合时）。**公赋《南山有台》**（"乐只君子，邦家之基"——夸武子是国家柱石）。武子**去所**："臣不堪也。"
→ 可援引：报聘、复命两个场合的配诗；臣受君誉的避席之谦。

### **16.〖襄公二十六年〗澶渊之后：两诗救一君**
卫侯被晋扣押。齐侯、郑伯为卫侯赴晋说情，晋侯兼享之。**晋侯赋《嘉乐》**（宾主尽欢开场）。国景子相齐侯**赋《蓼萧》**（"既见君子，为龙为光"——沾晋侯之光，小国颂霸主标准诗）。子展相郑伯**赋《缁衣》**（"适子之馆兮……敝予又改为兮"——郑对晋永远追随不贰）。叔向命晋侯拜二君。宴中晋侯言卫侯之罪，**国子赋《辔之柔矣》**（逸诗："马之刚矣，辔之柔矣"——驭诸侯当用软缰绳，劝宽），**子展赋《将仲子兮》**（"人之多言，亦可畏也"——扣押诸侯，舆论可畏），**晋侯乃许归卫侯**。
→ 可援引：多国联合斡旋释放人质的全套流程；软硬两诗配合施压。

### **17.〖襄公二十七年〗垂陇之享：七子赋诗，赵孟观志**
郑简公享赵孟于垂陇，七卿从。赵孟："请皆赋以卒君贶，武亦以观七子之志。"子展赋《草虫》→"善哉！民之主也"；伯有赋《鹑之贲贲》→"床笫之言不逾阈……非使人之所得闻也"；子西赋《黍苗》之四章→"寡君在，武何能焉"；子产赋《隰桑》→"武请受其卒章"（卒章"中心藏之，何日忘之"——把你的教诲藏在心里）；子大叔赋《野有蔓草》→"吾子之惠也"；印段赋《蟋蟀》→"善哉！保家之主也"；公孙段赋《桑扈》→"'匪交匪敖'，福将焉往？"卒享，赵孟告叔向："伯有将为戮矣！诗以言志，志诬其上而公怨之……幸而后亡。"（数年后伯有果死于内乱。）
→ 可援引：观志术全景；每首诗一句定评的"赵孟体"；评赋模式的母本。

### 18.〖襄公二十七年〗楚薳罢莅盟——《既醉》
楚薳罢如晋莅盟，晋将享之，将出，**赋《既醉》**（"既醉以酒，既饱以德"——谢主之德，不忘君命）。叔向据此预言："薳氏之有后于楚国也，宜哉！承君命，不忘敏。"
→ 可援引：一首诗换一句家族国运预言；答谢享礼的标准诗。

### 19.〖襄公二十八年〗叔孙穆子食庆封——《相鼠》《茅鸱》之骂
齐庆封（弑君集团当国者）奔鲁，叔孙穆子设食，庆封氾祭（礼仪粗鄙）。穆子不悦，**使工为之诵《茅鸱》**（逸诗，刺不敬），**亦不知**。（按《襄公二十七年》传，此前叔孙豹已为赋《相鼠》——"人而无仪，不死何为"——庆封亦弗知。）既而奔吴，后终被楚执杀。
→ 可援引：以诗当面骂人而对方无学不觉；"弗知"作为人物已死亡的文学判词。

### 20.〖襄公二十九年〗荣成伯赋《式微》——劝归
鲁襄公在楚滞留（适逢楚康王丧），季武子在国内趁机夺卞邑，公惧不敢归。**荣成伯赋《式微》**（"式微式微，胡不归？"），**公乃归**。
→ 可援引：一句诗解决"回不回国"的决策；诗作行动指令。

### **21.〖襄公二十九年〗季札观乐——听风辨国运**
吴公子季札聘鲁，请观周乐。使工为之歌《周南》《召南》→"始基之矣，犹未也，然勤而不怨矣"；《邶》《鄘》《卫》→"忧而不困者也……是其《卫风》乎"；《王》→"思而不惧，其周之东乎"；《郑》→"其细已甚，民弗堪也，**是其先亡乎**"；《齐》→"泱泱乎！大风也哉！……国未可量也"；《豳》→"乐而不淫"；《秦》→"此之谓夏声……其周之旧乎"……
→ 可援引：从乐风推政风国运的完整方法论；十五国风的"外交人格"一览。

### 22.〖昭公元年〗虢之会后：令尹之志
楚令尹围（后之楚灵王）享赵孟，**赋《大明》之首章**（"明明在下，赫赫在上"——文王受命之诗，僭越自比）。**赵孟赋《小宛》之二章**（"各敬尔仪，天命不又"——警告天命不再来）。事毕赵孟问叔向："令尹自以为王矣，何如？"叔向："王弱，令尹强，其可哉！虽可，不终……不义而强，其毙必速。"
→ 可援引：僭越信号的识别；以诗警告的克制答法；预言推理链。

### **23.〖昭公元年〗郑伯兼享赵孟——《瓠叶》定一献**
子皮筹备享赵孟，**赵孟先赋《瓠叶》**（"幡幡瓠叶，采之亨之。君子有酒，酌言尝之"——薄酒小酌之诗），预告只受一献之礼。子皮原备五献，穆叔点破："夫人之所欲也，又何不敢？"及享，具五献笾豆于幕下，赵孟辞，乃用一献。宴中**穆叔赋《鹊巢》**（"维鹊有巢，维鸠居之"——晋君之功，子之力也）→"武不堪也"；**又赋《采蘩》**（"于以采蘩……公侯之事"——小国如蘩菜，供大国采用即安）→ 赵孟曰："小国为蘩，大国省穑而用之，其何实非命？"；**子皮赋《野有死麇》之卒章**（"舒而脱脱兮，无使尨也吠"——请从容缓进，勿使犬吠惊扰）；**赵孟答《常棣》**（"兄弟阋于墙，外御其务"）且曰："吾兄弟比以安，**尨也可使无吠**。"穆叔、子皮及曹大夫兴拜举爵："小国赖子，知免于戾矣。"
→ 可援引：以诗预告礼数规格；情诗断章与妙答连环；小国—大国互文的完整台词库。

### 24.〖昭公二年〗韩宣子聘鲁、聘卫——观书赋诗之旅
韩宣子来聘观书于大史氏，叹"周礼尽在鲁矣"。公享之：**季武子赋《绵》之卒章**（"予曰有奔奏，予曰有御侮"——晋侯有你这样的辅弼），**韩子赋《角弓》**（"兄弟婚姻，无胥远矣"——晋鲁亲如兄弟）。武子拜："敢拜子之弥缝敝邑，寡君有望矣。"**赋《节》之卒章**（《节南山》"式讹尔心，以畜万邦"——望你化育万邦）。宴于季氏，有嘉树，宣子誉之，武子："宿敢不封殖此树，以无忘《角弓》！"**遂赋《甘棠》**（召公遗爱之树——你就是召公）。宣子："起不堪也，无以及召公。"后聘卫，**北宫文子赋《淇澳》**（即《淇奥》，"有匪君子，如切如磋"——美武公之德颂宣子），**宣子赋《木瓜》**（"投我以木瓜，报之以琼琚"——必有厚报）。
→ 可援引：一次聘问全程多站赋诗的连续剧本；借园中实物（嘉树）串联诗典的高级技巧。

### 25.〖昭公三年〗楚灵王享郑伯——《吉日》即请猎
十月郑伯如楚，子产相。楚子享之，**赋《吉日》**（天子田猎之诗："吉日维戊，既伯既祷"）。既享，**子产乃具田备**，王以田江南之梦。
→ 可援引：诗即日程安排；听诗办事的相礼之能。

### 26.〖昭公十二年〗宋华定聘鲁——不知《蓼萧》，断为必亡
宋华定来聘，享之，**为赋《蓼萧》**（"既见君子……为龙为光"，宠光同福之颂）。**弗知，又不答赋**。昭子断："必亡。宴语之不怀，宠光之不宣，令德之不知，同福之不受，将何以在？"（华定后果奔亡。）
→ 可援引：不学之默的死刑判词；评赋反面教材第一案。

### **27.〖昭公十六年〗郑六卿饯韩宣子——"赋不出郑志"**
韩宣子为玉环事求郑（子产刚以大义拒献玉），郑六卿饯之于郊。宣子："二三君子请皆赋，起亦以知**郑志**。"子齹赋《野有蔓草》（喜遇君子）→"孺子善哉！吾有望矣"；**子产赋郑之《羔裘》**（"彼其之子，舍命不渝"——郑对晋忠贞，特标"郑之"以别唐、桧同名篇）→"起不堪也"；**子大叔赋《褰裳》**（"子不我思，岂无他人？"——晋若不顾郑，郑自有他国可投！）→ 宣子："起在此，敢勤子至于他人乎？"子大叔拜。宣子："善哉，子之言是！不有是事，其能终乎？"子游赋《风雨》（"既见君子，云胡不喜"）、子旗赋《有女同车》（"洵美且都"）、子柳赋《萚兮》（"倡予和女"——你唱我和）。宣子喜："郑其庶乎！……**赋不出郑志，皆昵燕好也**。"皆献马，**而赋《我将》**（"日靖四方"——我当保郑安宁）。子产拜，使五卿皆拜："吾子靖乱，敢不拜德！"
→ 可援引：饯行观志名场面；《褰裳》式"以别选项施压"的合法威胁；软（五卿言好）硬（一卿亮牌）搭配的团队战术。

### 28.〖昭公十七年〗小邾穆公来朝——《采菽》《菁菁者莪》
小邾穆公朝鲁，公与之燕。**季平子赋《采叔》**（即《采菽》，天子所以赐诸侯——以鲁拟天子格局待客，暗含季氏之骄）。**穆公赋《菁菁者莪》**（乐见君子）。昭子评："不有以国，其能久乎？"
→ 可援引：赋诗背后的僭越气息可被旁观者记账。

### 29.〖昭公二十五年〗宋公享昭子——乐极之悲
宋元公享叔孙昭子，**赋《新宫》**（逸诗，盖颂新庙），**昭子赋《车辖》**（即《车舝》，"式燕且喜"迎娶燕乐之诗——此行兼为季平子迎宋女）。明日宴饮酒乐，宋公使昭子右坐，语相泣也。乐祁退而告人："今兹君与叔孙其皆死乎？吾闻之：**哀乐而乐哀，皆丧心也**。"（是岁二人皆卒。）
→ 可援引：宴饮情态本身也是"志"，失度即凶兆；逸诗《新宫》实例。

### **30.〖定公四年〗申包胥哭秦庭——《无衣》出师**
吴破楚入郢，申包胥如秦乞师，辞曰："吴为封豕、长蛇，以荐食上国……"秦伯使辞："子姑就馆，将图而告。"对曰："寡君越在草莽，未获所伏，下臣何敢即安？"**立依于庭墙而哭，日夜不绝声，勺饮不入口七日。秦哀公为之赋《无衣》**（秦风："岂曰无衣？与子同袍。王于兴师，修我戈矛，与子同仇！"），**九顿首而坐，秦师乃出**。
→ 可援引：赋诗时代的压轴绝唱；诗即出兵令；"哭庭七日+一诗定盟"的极限乞师范式。

## 乙编：源头与边缘

赋诗之源是"民之歌之"与宫廷乐语，以下 6 事非典型外交，但常被引为背景：

31.〖闵公二年〗**许穆夫人赋《载驰》**——卫亡，夫人驰归吊唁抒愤，是《载驰》本事（后世外交断章取"控于大邦"义即源于此篇）。同年**郑人为之赋《清人》**——高克之师溃散，国人作诗刺郑文公弃师，"赋"在此谓创作。
32.〖隐公三年〗**卫人所为赋《硕人》**——庄姜美而无子，国人闵之而作。"赋"=创作的最早用例。
33.〖文公六年〗**秦人哀三良，为之赋《黄鸟》**——殉葬激起国人作诗，君子据此断"秦之不复东征也"。
34.〖襄公十四年〗**卫献公使师曹歌《巧言》之卒章**（"彼何人斯？居河之麋。无拳无勇，职为乱阶"——影射孙文子据河边采邑将作乱）——本欲羞辱，反促孙文子先发动叛乱，献公出奔。以诗辱人致祸的头号反面教材。
35.〖昭公二十五年〗宋公享昭子事已入甲编 29；另**哀公十一年公孙夏命其徒歌《虞殡》**（送葬歌）——齐师赴艾陵之战自知必死，军歌明志。
36.〖成公九年〗晋侯观军府见楚囚钟仪，**使与之琴，操南音**——范文子评"乐操土风，不忘旧也"，因此建议放钟仪归楚促和。音乐即国籍与心迹的证明，与季札观乐同一原理。
