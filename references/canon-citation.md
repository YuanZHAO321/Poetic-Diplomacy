# 引经据典：《书》《诗》的政治论证层

赋诗是表达，引经是论证。本文档覆盖《左传》全书 **43 处引《书》、126 处"《诗》曰/云"** 的穷举成果——这是辞令（散文外交）里"证明某种政治原则的依据和正当性"的军火库。撰写辞命、谏辞、盟会争议、评赋预言时用。

## 一、三层语料结构

> 赵衰曰："说礼乐而敦《诗》《书》。**《诗》《书》，义之府也**；礼乐，德之则也。"（僖二十七）——春秋人自己的定位。

| 层 | 动词 | 功能 | 语义规则 |
|---|---|---|---|
| **赋《诗》** | 赋 | 外交表达（表演性言语行为） | 断章取义合法，义随场合流动 |
| **引《诗》** | 《诗》曰/云 | 辞令中的情理论据、收束咏叹 | 取义须贴近诗句本旨 |
| **引《书》** | 《书》曰/《周书》曰… | 法理论据：先王成宪、政治原则 | **不可断章**——引它就是引宪法，义必忠实 |

《书》从不被"赋"：没有人在宴会上诵《康诰》，但论证"你不该这么做"时，"《周书》曰"一出即援引了双方共认的最高法源。引证效力等级：先王之命 ≈ 引《书》 > 盟府载书 > 对方先君德政 > 引《诗》 > 谚语、《志》。

## 二、引《书》军火库（按政治原则分类）

体例：引文 → 今本出处（**今文**＝可靠西周文献；**伪古文**＝晚出篇，实为后人反向抄撮《左传》，引用时只标《周书》即可，勿倒果为因；**逸书**＝今本不存）→《左传》战例。

### 甲、德命论（政权合法性的总纲）

| 引文 | 今本 | 战例 |
|---|---|---|
| **皇天无亲，惟德是辅** | 伪古文《蔡仲之命》 | 宫之奇谏假道（僖五）：连引三条《周书》驳"神必据我"——"鬼神非人实亲，惟德是依"。虞公不听，国亡 |
| 黍稷非馨，明德惟馨 | 伪古文《君陈》 | 同上，第二证 |
| 民不易物，惟德繄物 | 伪古文《旅獒》作"人不易物，惟德其物" | 同上，第三证。一段谏辞三引《书》，论证密度的天花板 |
| **惟命不于常** | 今文《康诰》 | 范文子鄢陵战胜当日警告晋厉公"有德之谓"（成十六）；君子论庆氏之亡（襄二十三）。天命随德转移——对胜利者最刺耳也最必要的话 |
| 明德慎罚 | 今文《康诰》 | 申公巫臣谏楚庄王勿纳夏姬（成二）："文王所以造周也"——以开国宪章拦住国君私欲 |
| 不敢侮鳏寡 | 今文《康诰》 | 韩厥为赵氏存后（成八）："三代之令王，皆数百年保天之禄"，说动晋侯立赵武 |

### 乙、民本与天命

| 引文 | 今本 | 战例 |
|---|---|---|
| **民之所欲，天必从之** | 伪古文《泰誓上》 | 子羽据以预言三国大夫将有忧（昭元）；穆叔据以预言鲁襄公"必死是宫"（襄三十一）。《左传》引用率最高的《大誓》句 |
| 纣有亿兆夷人，亦有离德；余有乱臣十人，同心同德 | 伪古文《泰誓中》 | 苌弘答刘子（昭二十四）："此周所以兴也。君其务德，无患无人" |
| 商兆民离，周十人同 | 同上（约文） | 君子论鞌之战"众之不可以已"（成二） |

### 丙、刑罚法理（判例引证的高峰）

| 引文 | 今本 | 战例 |
|---|---|---|
| **与其杀不辜，宁失不经** | 伪古文《大禹谟》 | 声子说楚复伍举（襄二十六）："赏僭则惧及淫人，刑滥则惧及善人"——疑罪从无的最早成文表述，直接促成人才回流的外交议题 |
| **父子兄弟，罪不相及** | 逸书（托《康诰》） | 苑何忌辞齐侯之赏（昭二十）："与于青之赏，必及于其罚"——以反连坐原则拒绝利益捆绑 |
| 父不慈，子不祗……不相及也 | 逸书（托《康诰》） | 臼季举郤缺（僖三十三）：其父有罪不掩其子之贤，"舜之罪也殛鲧，其举也兴禹" |
| 昏、墨、贼，杀 | 逸书（"皋陶之刑"） | 叔向断邢侯之狱（昭十四），大义灭亲并罪三人，孔子赞"古之遗直" |
| 其有颠越不共，则劓殄无遗育 | 今文《盘庚中》（作"颠越不恭"） | 伍子胥引《盘庚之诰》谏吴王灭越（哀十一）："是商所以兴也。今君易之，将以求大，不亦难乎？"不听，吴亡 |
| 毁则为贼，掩贼为藏……在《九刑》不忘 | 逸书《誓命》 | 大史克论逐莒仆（文十八）：整段辞令即一篇罪名定义学 |

### 丁、居安思危与慎终（谏诤常备）

| 引文 | 今本 | 战例 |
|---|---|---|
| **居安思危** | 逸书 | 魏绛辞晋悼公之乐（襄十一）："思则有备，**有备无患**，敢以此规"——引《书》辞赏的典范，悼公答"子之教，敢不承命" |
| 慎始而敬终，终以不困 | 逸书（伪古文《蔡仲之命》"慎厥终，惟其始"盖本此） | 大叔文子预言宁喜之灭（襄二十五）："置君而弗定……九世之卿族，一举而灭之" |
| 怨岂在明？不见是图 | 伪古文《五子之歌》 | 单子论郤至必亡（成十六）："将慎其细也。今而明之，其可乎？" |
| 恶之易也，如火之燎于原，不可乡迩 | 今文《盘庚上》（作"若火之燎于原"） | 君子论陈桓公长恶不悛（隐六）、蔡哀侯自取灭亡（庄十四）——两引，除恶务早的定谳之辞 |
| 欲败度，纵败礼 | 伪古文《太甲中》 | 子皮自省（昭十）："夫子（子产）知度与礼矣，我实纵欲而不能自克也" |

### 己、兼并与国际关系"法理"

| 引文 | 今本 | 战例 |
|---|---|---|
| **乱者取之，亡者侮之。推亡固存，国之利也** | 《仲虺之志》（今本《仲虺之诰》存"推亡固存，邦乃其昌"） | 子皮论伯有之败（襄三十）。这是春秋兼并战争最常援引的"合法性条款"——内乱之国可取，将亡之国可侮 |
| 殪戎殷 | 今文《康诰》 | 中行桓子谏缓伐赤狄（宣六）："使疾其民，以盈其贯，将可殪也"——引文王灭殷的成例论"等它恶贯满盈" |
| 官占，唯能蔽志，昆命于元龟 | 伪古文《大禹谟》 | 君子赞楚惠王"宁如志，何卜焉"（哀十八）——人谋先于龟卜 |

另：《郑书》"安定国家，必大焉先"（子产引，襄三十）是**郑国国史**而非《尚书》；《志》《军志》《前志》同属史志类引证，等级低于《书》。

## 三、引《诗》作论据（126 处的用法规律）

引《诗》与赋《诗》完全不同：它出现在散文辞令和评论内部，功能是**情理升华与收束定谳**，标准句式是——

> 《诗》曰："……"。**其（某）之谓矣／此之谓也／其是之谓乎！**

高频引句（出现两次以上者，皆经原文统计）：

| 引句 | 出处 | 论证用途 |
|---|---|---|
| 靡不有初，鲜克有终 | 《大雅·荡》 | 预言人物不得善终（宣二引于赵盾谏晋灵公；襄三十一两引） |
| 人之云亡，邦国殄瘁 | 《大雅·瞻卬》 | 哀贤人之失，论国将衰（文六秦穆殉良、襄二十六声子论楚材晋用等三引） |
| 不解于位，民之攸塈 | 《大雅·假乐》 | 论恪守职位（成二、昭二十一、哀五三引） |
| 战战兢兢，如临深渊，如履薄冰 | 《小雅·小旻》 | 论戒惧自保（僖二十二、宣十六两引） |
| 君子屡盟，乱是用长 | 《小雅·巧言》 | 论盟誓泛滥无益（桓十二、襄二十九两引）——对"会盟外交"的内部批判 |
| 孝子不匮，永锡尔类 | 《大雅·既醉》 | 论孝德感化（隐元颍考叔"纯孝"、成二两引） |
| 岂不夙夜，谓行多露 | 《召南·行露》 | 论慑于形势不敢行（僖二十、襄七等三引） |
| 赫赫宗周，褒姒灭之 | 《小雅·正月》 | 叔向论"强不义"必亡（昭元，紧接令尹赋《大明》事）——引诗与评赋联动的实例 |

写作时的搭配公式：**历史先例开路（"昔者……"）→ 引《书》立法理 → 引《诗》动情理 → "其是之谓乎"收束**。宫之奇、声子、魏绛诸辞皆此结构。

## 四、写作规则（引经四诫）

1. **引《书》必核原文**：`python3 scripts/find_shu.py --grep 明德慎罚`。《左传》引文与今本常差一二字（如↔若、弗↔不、人↔民、懋↔茂），查不到先换异文字。
2. **分清文献层**：引文仅见于伪古文篇（大禹谟、泰誓、太甲、蔡仲之命等二十五篇）时，创作中只称"《周书》曰／《夏书》曰"（这正是《左传》原貌），**不要写出伪古文篇名**——那些篇名是东晋才有的。
3. **逸书照引不误**："居安思危""父子兄弟罪不相及"等虽今本不存，春秋人引之凿凿，创作中可径引并在注解里标"逸书"。
4. **引《书》不断章**：赋《诗》可以移花接木，引《书》若歪曲原义，等于伪造法条——对方大夫当场就能识破，反成笑柄。
