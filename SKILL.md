# 赋诗言志：春秋古外交

春秋时代（本技能覆盖《左传》闵公二年至定公四年，约公元前 660–前 506 年），列国卿大夫在聘问、盟会、享宴、饯行等外交场合不直陈其事，而是诵《诗经》某篇某章，借诗句含蓄传达立场——这就是"赋诗言志"。一首诗赋得恰当可以借来军队（秦哀公赋《无衣》而出师）、救回国君（子展赋《将仲子》而晋侯归卫侯）；赋得不当则当场结怨甚至亡身（伯有赋《鹑之贲贲》，赵孟预言"将为戮矣"）。它是一套有明确规则、有生死利害的信号系统，不是风雅点缀。

## 任务模式

先判断用户要什么，再读对应参考文档：

| 模式 | 用户诉求 | 必读 |
|---|---|---|
| **拟赋** | 有一个外交意图（求援、修好、施压、拒绝、致谢…），要挑合适的诗并设计应对 | `references/repertoire.md` + `references/protocol.md` |
| **解赋** | 已知某人在某场合赋了某诗，要解读用意、判断该如何答赋 | `references/repertoire.md` + `references/episodes.md` |
| **演礼** | 要一整场聘问/盟会/享宴/饯行的完整场景或剧本 | 全部四份参考 |
| **评赋** | 评判一次赋诗是否合礼（类不类、僭不僭），并像赵孟、叔向那样据诗观志、预言吉凶 | `references/protocol.md` + `references/episodes.md` |
| **辞命** | 需要散文体外交辞令（乞师、绝交、犒师、答问、谏辞）或《尚书》体誓、诰、命 | `references/rhetoric.md` + `references/canon-citation.md` |

现代场景转译（把谈判、商务邮件、发言稿写成赋诗外交）属于拟赋或演礼：先把现代意图映射为春秋意图（合作=修好，催款=责贡，挖角=褰裳之讽），再照常操作。

## 工作流程

1. **定场合与身份**。赋诗只发生在特定礼仪时点：享礼"礼终乃宴"之后、饯行郊外、乞师庭中。先确定双方国别、爵位（君/卿/大夫）、宾主关系、大国小国之势——这决定哪些诗能用（见等级限度）。
2. **引经前必查原文**。凡引《诗经》《尚书》，先运行：
   ```bash
   python3 scripts/find_poem.py 褰裳            # 按篇名查《诗经》全文
   python3 scripts/find_poem.py --grep 同袍     # 按诗句反查篇目
   python3 scripts/find_shu.py 康诰             # 按篇名查《尚书》（自动标注今文/伪古文）
   python3 scripts/find_shu.py --grep 明德慎罚  # 按句反查所在篇
   ```
   赋诗以逐字精确为生命，错一字即失礼。同名篇必须按国风分辨（《无衣》有唐、秦两篇，意义天差地别；子产赋诗特称"郑之《羔裘》"就是在消歧）。《左传》所记篇名与今本《诗经》有异文与逸诗，对照表见 `references/repertoire.md` 末尾——查不到的先查异名，再判为逸诗。引《书》的异文与真伪层规则见 `references/canon-citation.md`。
3. **断章**。选定篇后再选章："赋《载驰》之四章"只取"控于大邦"一句之义，与全诗本义无关。指明取哪一章、取哪一句、借它说什么。
4. **设计应答链**。赋诗是对话：主赋—宾答—拜谢，每一环都有含义（包括沉默）。写清对方可能如何接、我方如何收尾（拜、降拜、稽首，或"不堪也"之谦）。
5. **核对先例**。给出的每个方案，尽量援引 `references/episodes.md` 中的《左传》先例佐证（某年某人曾以此诗达成此事），这是方案说服力的来源。

## 核心规则速览

完整规则与原文出处见 `references/protocol.md`。

1. **断章取义是正当操作**——"赋诗断章，余取所求焉"（襄二十八）。情诗《野有蔓草》"邂逅相遇，适我愿兮"在外交中只表"喜遇嘉宾"，无人会想到男女之情。
2. **歌诗必类**——诗与场合、身份、意图必须相称。温之会齐高厚"歌诗不类"，当场被判"有异志"，诸侯同盟讨之（襄十六）。
3. **等级有限度**——《肆夏》天子享元侯、《文王》两君相见、《湛露》《彤弓》天子宴诸侯：卿大夫遇之当辞不敢当（穆叔三拜《鹿鸣》而不拜其上，襄四；甯武子佯不知以拒僭礼，文四）。小国之卿受大国之君过高礼遇，答语是"臣不堪也"。
4. **答赋是义务，沉默有语义**——不答赋或为无学（宋华定不知《蓼萧》，断为"必亡"，昭十二），或为谦让（"武不堪也"），或为守礼拒收（甯武子）。设计场景时必须区分。
5. **赋诗即承诺**——诗一出口等于口头条约：秦哀公赋《无衣》即是出兵令（定四）；叔孙穆子赋《匏有苦叶》，叔向退而备船（襄十四）；楚灵王赋《吉日》，子产连夜备田猎之具（昭三）。
6. **诗以观志**——听者据诗判断赋者心术乃至国运：赵孟评垂陇七子，一一预言其家族存亡（襄二十七）；令尹围赋《大明》首章自比文王，叔向断其"必毙"（昭元）。
7. **称谓要地道**——自称"寡君（我国君）、敝邑、下臣、不腆"，敬称"君、执事、吾子"，套语"敢拜、敢不承命、辱贶、其敢干大礼以自取戾"。
8. **赋《诗》可断章，引《书》不可**——散文辞令中"《书》曰""《诗》曰"是论证行为不是表演行为：《书》是双方共认的法源（"《诗》《书》，义之府也"，僖二十七），歪曲原义等于伪造法条。论证层的引证等级、军火库与句式见 `references/canon-citation.md`。

## 输出格式

**拟赋**给方案表：场合时点 → 所赋篇章（原文引全该章）→ 断章取义说明 → 预期答赋与礼节 → 《左传》先例 → 风险（何种情况下会被解读为失礼/僭越）。若有多案，按稳妥到激进排序。

**解赋**给判读：原诗该章全文 → 表层义/断章义 → 结合局势的真实意图 → 三种可选回应（各配答赋篇目与后果推演）。

**演礼**写成剧本体：仪程标注（郊劳/享/宴/饯）、人物言辞用文言、赋诗处引原文全章、旁注每一步的用意与礼数，结尾附"君子曰"式评点。

**评赋**学赵孟体例：逐一"某赋《某》，评曰……"，判类不类、僭不僭，据诗观志，给出预言并说明推理（诗中何句泄露何心）。

**辞命**依 `references/rhetoric.md` 的文体格式：乞师学申包胥，绝交学吕相，犒师学展喜，誓师用《尚书》誓体三段式。论证骨架用 `references/canon-citation.md` 的搭配公式：历史先例开路 → 引《书》立法理 → 引《诗》动情理 → "其是之谓乎"收束。

## 红线

- 引诗必须经 `find_poem.py` 核实、引《书》必须经 `find_shu.py` 核实，不得凭记忆默写；逸诗（《河水》《茅鸱》《新宫》《辔之柔矣》《狸首》等）与逸书（"居安思危""父子兄弟罪不相及"等）明确标注，只可转述《左传》所记文句与用意，不得伪造扩写。创作中引伪古文篇内容时只称"《周书》曰／《夏书》曰"，不出现东晋才有的篇名。
- 不发明《左传》没有的"先例"。可以虚构新场景（演礼本来就是创作），但援引历史时年份、人物、篇目须与 episodes.md 一致。
- 人物身份与时代不错位：不要让重耳引用他死后才发生的典故，不要让大夫行天子之礼而无人纠正——若剧情需要失礼，须让场中人像叔向、赵孟那样点破，这正是戏眼。
